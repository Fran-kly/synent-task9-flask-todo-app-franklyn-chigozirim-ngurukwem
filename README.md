Flask To-Do App (Full-Stack)

A full-stack To-Do web application built with Flask, featuring user authentication and database integration.

Features

- User Registration & Login (Authentication)
- Add tasks
- Mark tasks as completed
- Delete tasks
- Tasks saved per user
- Clean and responsive UI

Tech Stack

- Python (Flask)
- SQLite (Database)
- HTML & CSS

How to Run

1. Install dependencies:
   pip install flask flask_sqlalchemy

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000

Project Structure

- app.py → Main backend logic
- templates/ → HTML pages
- database.db → SQLite database (auto-created)
